# angular7-example

### Details

The project is built using Angular 7.
It uses OMDB Api, the code for it can be found in `app.service.ts`.

### Setup

For debugging, 
```
npm install
ng serve
```

For live,
```
npm install
npm run postinstall
npm run start
```
